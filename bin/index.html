Overview

This is a free time project. This program is intended for people living in countries where their rights are violated, for whistleblowers and for journalists.

Features

  • simple but strong peer-to-peer connexion, no servers involved
  • highly secure symmetric encryptions methods
  • simple and easy to use GUI
  • further development (this is only the beginning)
  • configurable receive and send ports and ip
  • automatic check for updates
  • automatic port forwarding using uPnP

Installation

Install QuarkChat

QuarkChat is a portable program, so no install is needed.

Configuration

For this version 0.1 (beta), you are able to change the connexion settings and encryption settings

Encryption and Connexion

Encryption

For this version only one Symmetric and no Asymmetric Encryption are implemented.

Name description
AES encryption of electronic data established by the U.S. National Institute of Standards and Technology (NIST) in 2001

Connexion

This software is using pure peer-to-peer connexion. For NAT connexions, it will automatically program the port forwarding of NAT using uPnP tehnology.

Further Development

This is the first published version of this software, in the following versions we will add strong asymmetric and symmetric encryption, VoIP and fingerprint check system. Also we will try to implement as fast as we can uPnP connexion.

References

We used WAIFUPNP for uPnP.
